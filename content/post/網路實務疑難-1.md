1. DHCP 避讓 **Static ARP (靜態 ARP)** 、**DHCP 位址保留 (DHCP Reservation)**
很多廠商（特別是中低階設備）是把 **DHCP Server 模組**與 **ARP Table 模組**分開寫的
交換器端：手動設定固定 IP（例如 192.168.1.2），遮罩 255.255.255.0，閘道指向路由器 192.168.1.1。
路由器端 (TP-Link)：進入 DHCP 設定，將發放範圍（Pool）修改為 .100 到 .254。
結果：.2 到 .99 變成「手動保留區」，DHCP 永遠不會去動這區，自然不會有衝突。

ASUS
**網路 / 主機 IP**`10.0.0.0`告訴路由器：這是你要找的 PVE SDN **目的網段**（後面的 `.0` 代表整個網段）。若你的 SDN 是 `10.1.1.x`，這裡就填 `10.1.1.0`。
**網路遮罩** `255.255.255.0`通常 SDN 的子網路遮罩預設是 `/24`，對應成 IP 格式就是 `255.255.255.0`。
**閘道器** `192.168.8.100`**關鍵！** 請填入你的 **PVE 主機在實體家裡的 IP**。這代表當有封包要去 `10.x` 時，ASUS 路由器會把封包丟給這台 PVE 主機，由它帶路送進 SDN。**順序 (Metric)**`1` 或 `10`路由的優先權，通常填寫 `1` 或 `10` 即可（數字越小優先度越高）。
