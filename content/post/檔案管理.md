#### TotalCommander + Beyond Compare
自訂功能列在`Wincmd.ini` 中找到 \[Configuration\]加入
```
[Configuration]
CompareTool=c:\ProgramFiles\BeyondCompare4\BCompare.exe
StartupScreen=0
LanguageIni=wcmd_tw.lng
Mainmenu=wcmd_tw.mnu
UseRightButton=1
SizeFooter=3
MarkDirectories=1
SortDirsByName=0
ShowHiddenSystem=0
```

### Everything 設置
1. AHK設定搜尋特定位置
```
!Space::
Run, "C:\Program Files\Everything\Everything.exe" -filename E:\desktopz\everythingJump
Return
```
2. 鎖定搜尋窗口大小
在1.5.0.1257a之後版本，搜尋框輸入`/window_style_remove=0x40000`後按Enter，要解鎖搜尋框輸入`/window_style_remove=`
3. 調用TotalCMD開啟
`Everything.ini` 修改
```
open_folder_path_command=$exec("c:/Program Files/totalcmd/TOTALCMD.EXE" "/O" "/T" "$parent(%1)")
open_folder_command=$exec("c:/Program Files/totalcmd/TOTALCMD.EXE" "/O" "/T" "%1")
```
