**啟動選項**    
You can specify the following command line parameters at startup of CLaunch.  

| 參數      | 說明                                                                                                                                                                                                                                                                    |
| ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **/n**  | Starts CLaunch in non-resident mode.  <br>The window appears on startup and exits when you close the window.  <br>By default, it is initially displayed at the position when CLaunch was last closed, but you can change the initial position by specifying /m or /s. |
| **/nh** | Starts CLaunch in the [unhooked](extSuspend.htm) state.                                                                                                                                                                                                               |
| **/l**  | Displays CLaunch in pinned state.  <br>Initial display position will be last closed position of CLaunch by default, but you can specify the display position with combining the /m or /s.                                                                             |
| **/m**  | Displays CLaunch to the position of the mouse cursor.                                                                                                                                                                                                                 |
| **/s**  | Displays CLaunch to the specified coordinates.  <br>**Format** : /s[X coordinate]:[Y coordinate]  <br>**Example** : /s200:100                                                                                                                                         |
| **/p**  | Specifies the initial display page.  <br>**Format** : /p[Page Number]  <br>**Example** : /p2                                                                                                                                                                          |
| **/b**  | Specifies the number of buttons.  <br>**Format** : /b[Horizontal]:[Vertical]  <br>**Example** : /b5:3                                                                                                                                                                 |
| **/d**  | Specifies the wait time to delay the start of CLaunch in millisecond.  <br>**Format** : /d[Wait Time]  <br>**Example** : /d3000 (Wait 3 seconds)                                                                                                                      |
| **/a**  | Enables the auto-hide function.  <br>After CLaunch is displayed by /n, /l, /m or /s, it will automatically hidden when the cursor moves outside the window.                                                                                                           |
| **/r**  | Discards the cache data and retrieve the icon images.                                                                                                                                                                                                                 |
**Multiplex Startup of CLaunch**  

If CLaunch is already resident and another process of CLaunch.exe is launched, the window of first process will be displayed. (Forcibly displayed even in Restricted Mode or unhooked.)  
At this time, the command line parameters **/l, /m, /s, /p, /a** can be specified to the second CLaunch.
### **範例批次檔代碼**

以下代碼將讀取 `exe_paths.txt` 文件中的每個 `.exe` 路徑，並在指定的目標資料夾中創建快捷方式。
1. 打開記事本，並將以下代碼複製到新文件中。  
2. 將文件另存為 `.bat` 文件，例如 `CreateShortcutsFromTxt.bat`。

```batch
@echo off
setlocal enabledelayedexpansion

:: 設定包含 .exe 路徑的 txt 文件路徑
set "exeList=exe_paths.txt"
:: 設定快捷方式存放的目標資料夾
set "shortcutFolder=C:\Shortcuts"

:: 檢查目標資料夾是否存在，如果不存在則創建
if not exist "%shortcutFolder%" mkdir "%shortcutFolder%"

:: 遍歷 txt 文件中的每行
for /f "usebackq delims=" %%f in ("%exeList%") do (
    set "appPath=%%f"
    set "appName=%%~nxf"
    set "shortcutPath=%shortcutFolder%\!appName!.lnk"
    
    :: 使用 WScript 創建快捷方式
    echo Set oWS = CreateObject("WScript.Shell") > "%temp%\CreateShortcut.vbs"
    echo sLinkFile = "%shortcutPath%" >> "%temp%\CreateShortcut.vbs"
    echo Set oLink = oWS.CreateShortcut(sLinkFile) >> "%temp%\CreateShortcut.vbs"
    echo oLink.TargetPath = "%appPath%" >> "%temp%\CreateShortcut.vbs"
    echo oLink.Save >> "%temp%\CreateShortcut.vbs"
    
    :: 執行 VBS 腳本
    cscript //nologo "%temp%\CreateShortcut.vbs"
    
    :: 刪除 VBS 腳本
    del "%temp%\CreateShortcut.vbs"
)

echo 所有快捷方式已創建至 %shortcutFolder%。
endlocal
pause
```

### **解釋**

* `set "exeList=exe_paths.txt"`：指定包含所有 `.exe` 路徑的 `.txt` 文件。  
* `set "shortcutFolder=C:\Shortcuts"`：設置快捷方式的目標文件夾。  
* `for /f "usebackq delims=" %%f in ("%exeList%") do (...)`：逐行讀取 `.txt` 文件的內容，每一行作為一個 `.exe` 路徑。  
* `%%~nxf`：提取 `.exe` 文件名和擴展名來命名快捷方式。

運行這個批次檔後，`C:\Shortcuts` 資料夾中將包含 `exe_paths.txt` 文件中所有 `.exe` 文件的快捷方式。
