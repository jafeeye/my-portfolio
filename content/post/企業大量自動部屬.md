## IaC CT
```
[ 你電腦 Git Push ] ──► [ GitLab 專案 ] 
│ (發送任務) ▼
[ PVE LXC: ID 100 ] (長駐維運機) 
 ┌──────────────────────────────┐ 
 │ 1. GitLab Runner (收到任務)   │ 
 │ 2. Terraform (打 API 蓋 VM)   │──► [ PVE 宿主機 API ] 
 │ 3. Ansible (SSH 進 VM 裝軟體) │──► [ 新蓋好的 VM ] 
 └──────────────────────────────┘
```
1. 安裝Rocky Linux 的CT模板,在 https://images.linuxcontainers.org/images/ 可以找到或PVE裡面下載
2. 勾選 Unprivileged container （非特權容器）、nesting（巢狀虛擬化）
3. 因為是IaC專用機，把這台固定IP或實體路由器做DHCP Static IP Reservation或使用PVE IPAM
4. 安裝Terraform、Ansible、Gitlab Runner
```
:'inital'
dnf update -y && dnf install -y epel-release && dnf install -y curl wget git python3-pip openssh-clients ;

:'GitLab Runner'
curl -L "https://packages.gitlab.com/install/repositories/coordinator/script.rpm.sh" | bash && dnf install -y gitlab-runner ;

:'安裝 Terraform'
dnf config-manager --add-repo https://rpm.releases.hashicorp.com/RHEL/hashicorp.repo && dnf install -y terraform ; 

:'安裝 Ansible'
dnf install -y ansible-core ;

:'切換到 gitlab-runner 帳號，並生成 ed25519 金鑰（一路按 Enter 到底）'
su - gitlab-runner -c "ssh-keygen -t ed25519 -N '' -f ~/.ssh/id_ed25519" ;

# 印出公鑰，方便你等一下複製到 PVE 的 Cloud-Init 模板中
echo -e "\n=== 【 這是你的自動化 SSH 公鑰，請複製它 】 ===" && cat /home/gitlab-runner/.ssh/id_ed25519.pub ;
```
