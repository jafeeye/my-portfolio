Golden Image
```
Terraform
  ↓
PVE API
  ↓
Clone VM Template
  ↓
自動生成一個 CloudInit ISO
  ↓
掛到 VM
  ↓
Cloudbase-Init 讀取內容
```

進階流程
```
MDT
 ↓
建立 Golden Image
 ↓
Sysprep
 ↓
做成 PVE Template
 ↓
Terraform Clone
 ↓
Cloudbase-init
 ↓
Ansible
```

模板製作過程
```
Win11 VM
→ 安裝 VirtIO
→ 安裝 Cloudbase-Init
→ Sysprep
→ 關機
→ Convert to Template
```

優化項目
```
# 開啟rdp
Set-ItemProperty -Path 'HKLM:\System\CurrentControlSet\Control\Terminal Server' -name "fDenyTSConnections" -value 0  
# 關閉休眠
powercfg /hibernate off
# 關閉記憶體完整性
reg add "HKLM\System\CurrentControlSet\Control\DeviceGuard\Scenarios\HypervisorEnforcedCodeIntegrity" /v "Enabled" /t REG_DWORD /d 0 /f
# Credential Guard (停用會導致hyper-v無法開啟)

```

## Clonezilla 對拷至 PVE (P2V複製)
1. VM開機選擇Clonezilla 的ISO檔
2. 選擇`device-device` -> `remote`
3. 在一台實體機使用 Clonezilla 開機 `device-device` -> `remote`
4. 透過網路直接把實體機系統轉成VM
