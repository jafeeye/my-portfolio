Oracle Virtual Cloud Network (VCN)=Google Virtual Private Cloud (VPC)
OCI 的 VCN 是區域級（Regional）:無法選擇跨國家的
GCP的 VPC 是全球級（Global）
Oracle 設定2階
![](Pasted%20image%2020260519101901.png)

建立VM 步驟
1. VCN 建立 Networking/VCN，去建立CIDR Block 10.0.0.0/16 及subnet 10.0.0.0/24
2. VM建立 Compute/Instances，設定OS、網路、SSH連線後建立VM
3. 建立VM之後,一定要從SSH連進去設定初始帳號密碼,才能在Web上看到畫面(OS Management/Console connection/Cloud Run)
