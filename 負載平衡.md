## HA Proxy ＋ Keepalived 綁在一起？
大家會把這兩個放在一起，是為了追求「100% 的無單點故障（No Single Point of Failure）」。
- **只用單台 HAProxy：** 如果 Illumio Core 0 掛了，HAProxy 會把流量導向 Core 1（**沒問題**）。 但如果 **HAProxy 這台主機本身死機了**（例如硬體壞了或斷電），那就算後端的 Illumio Core 都活著，下方的 VEN 也全部連不上 PCE。
- **加上 Keepalived：** 你可以準備「兩台」HAProxy 主機，透過 Keepalived 創立一個虛擬 IP（VIP）。平常流量走第一台 HAProxy，第一台死機時，VIP 自動飄移到第二台 HAProxy。
